2️⃣ Global Alignment

Konsep: Menyelaraskan seluruh sequence dari awal hingga akhir. Semua karakter harus masuk alignment — gap disisipkan di mana perlu.

Algoritma: Needleman-Wunsch

Kapan digunakan: Ketika kedua sequence memiliki panjang serupa dan berasal dari region yang sama (misalnya: membandingkan gen ortolog).

1️⃣ Apa itu Gap?

Gap (-) adalah karakter fiktif yang disisipkan ke dalam sequence selama proses alignment agar dua sequence bisa disejajarkan secara optimal.

Gap merepresentasikan kejadian biologis nyata: insersi atau delesi (indel) yang terjadi selama evolusi.

seq1: ATGACC---TTAGC
seq2: ATGACCTGGTTAGC
             ^^^
         insersi di seq2 (atau delesi di seq1)

Gap Penalty = biaya komputasional yang dibayar setiap kali gap disisipkan.

  • Semakin besar penalti → algoritma semakin "hemat" dalam memasukkan gap
  • gap_open = penalti saat gap pertama kali muncul
  • gap_extend = penalti tambahan untuk setiap karakter gap berikutnya

3️⃣ Local Alignment

Konsep: Mencari subsequence terbaik dari dua sequence yang paling mirip. Tidak harus dari ujung ke ujung.

Algoritma: Smith-Waterman

Kapan digunakan: Ketika sequence panjang dan kita hanya peduli pada domain atau region yang conserved.

seq1: CGATGACAAGG--CATGACGATACGATAATAGCATGCA
seq2:      ATGACGAAATGACT--AAGCAAAGCAATGCA
                ^^^^^^^^
            region terbaik yang ditemukan

🔑 Decode Nama Fungsi pairwise2

Format: pairwise2.align.[scope][match][gap]

Posisi Kode Arti
Scope global Seluruh sequence
Scope local Subsequence terbaik
Match x match=+1, mismatch=0 (default)
Match m Custom match & mismatch score
Gap x Tidak ada penalti gap
Gap s Custom gap_open & gap_extend

Contoh: globalms = global scope + custom match + custom gap scoring

4️⃣ Hamming Distance

Definisi: Jumlah posisi yang berbeda antara dua sequence dengan panjang yang sama.

⚠️ Syarat mutlak: Panjang kedua sequence harus sama. Jika berbeda, Hamming Distance tidak terdefinisi — gunakan Levenshtein.

s1 = A C G T A C G T
s2 = A G G T A C C T
       ^           ^
   pos 2 beda   pos 7 beda
→ Hamming Distance = 2

5️⃣ Levenshtein Distance

Definisi: Jumlah minimum operasi edit yang dibutuhkan untuk mengubah string A menjadi string B.

3 Operasi yang diizinkan:

Operasi Contoh
Insertion AGT → ACGT
Deletion ACGT → AGT
Substitution ACGT → AGGT

Kelebihan vs Hamming: Levenshtein dapat menangani sequence dengan panjang berbeda

Perbandingan Hamming vs Levenshtein

Fitur Hamming Levenshtein
Panjang Sequence Harus sama Boleh berbeda
Operasi Substitution saja Insertion, Deletion, Substitution
Kompleksitas
Use Case Quick similarity check Edit distance umum

6️⃣ Dot Plot

Definisi: Metode visualisasi matrix 2D untuk membandingkan dua sequence. Titik/kotak ditandai jika karakter di posisi (i, j) sama.

Cara Membaca Dot Plot

Pola Interpretasi Biologis
Diagonal panjang & lurus Region sangat conserved (hampir identik)
Diagonal pendek/putus-putus Ada substitusi di region tersebut
Tidak ada pola diagonal Region sangat berbeda
Diagonal offset (tidak di tengah) Repeat region / tandem repeat ← insight terpenting

💡 Diagonal offset adalah kekuatan utama Dot Plot — fenomena ini tidak bisa terdeteksi dari angka Hamming atau Levenshtein saja.

Implementasi

def dotplot(s1, s2, nonblank=chr(0x25A0)):  # 0x25A0 = karakter ■
    print(" |" + s2)
    print("-" * (len(s2) + 2))

    for x in s1:
        line = ''
        for y in s2:
            if x == y:
                line += nonblank  # ■ jika karakter sama
            else:
                line += ' '       # kosong jika berbeda
        print(x + "|" + line)

dotplot(seq1, seq2)
